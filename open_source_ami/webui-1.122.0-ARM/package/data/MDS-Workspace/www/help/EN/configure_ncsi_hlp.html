NCSI

This page is used to configure NCSI configuration settings.

Actions

NCSI Interface

It lists the interface name in list box. Choose the particular Interface name to which you need to configure NCSI settings.

Channel Number

Choose the channel number to be configured for the selected Interface name.

Package ID

Choose the package ID to be configured for the selected Interface name.

Save

Click 'Save' to save the current changes.

Reset

Click 'Reset' to reset the modified changes.
