Dashboard

Dashboard displays the overall information about the device status.

Information

Device Information

Displays the Device Power Status(On or Off), Firmware Revision and Firmware Build Time (Date and Time).

Network Information

Shows network settings for the device. Click on the link Edit to view the Network Settings Page.

Remote Control

Start remote redirection of the host by launching the console from this page.

Java Console

Clicking on 'Launch' button of 'Java Console' will cause the jviewer.jnlp file to be downloaded. Once the file is downloaded and launched, a Java redirection window will be displayed.

Sensor Monitoring

It lists all available sensors on the device, with information such as status, name, reading, and status icon.
There are 3 possible states for a Sensor:
  • - Green dot denotes a Normal state.
  • - Yellow exclamation denotes Warning state.
  • - Red cross denotes a Critical state.
The magnifying glass allows access to the Sensor details page for that sensor.

Event Logs

A graphical representation of all events incurred by various sensors and occupied/available space in logs. If you click on the color-coded rectangle in the Legend for the chart, you can view a list of those specific events only.
