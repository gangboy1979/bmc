PAM Ordering

This page is used to configure the PAM order for user authentication into the BMC.

Actions

PAM Module

It shows the list of available PAM modules supported in BMC.

Move Up

Click on the required PAM module, it will be selected. Click on 'Move Up' option to move the selected PAM module one step before the existing PAM module.

Move Down

Click on the required PAM module, it will be selected. Click on 'Move Down' option to move the selected PAM module one step after the existing PAM module.

Save

Click 'Save' to save any changes made.
NOTE: Whenever the configuration is modified, the web server will be restarted automatically. Logged session will be logged out.

Reset

Click 'Reset' to reset the modified changes.
