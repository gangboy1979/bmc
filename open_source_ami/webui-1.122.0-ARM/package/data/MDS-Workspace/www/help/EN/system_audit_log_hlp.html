System & Audit logs

If configured, these logs will display all the system and audit events that occurred in this device.
NOTE: Logs have to be configured under 'Configuration -> System and Audit Log' in order to display any entries.

Actions

System Log

Click the System Log tab to view all system events. Entries can be filtered based on their levels like Alert, Critical, Error, Notification, Warning, Debug, Emergency and Information.

Audit Log

Click the Audit Log tab to view all audit events for this device.
