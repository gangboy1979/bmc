Restore Factory Defaults

This page will restore the device to default configuration. Press " Restore Factory Defaults" to restore the device default configuration.

WARNING: Please note that after entering into restore factory defaults, widgets, other web pages and services will not work. All open widgets will be closed automatically. The device will reset and reboot within few minutes.
This section lists the configuration items, that will be preserved during restore factory default configuration. Click "Preserve Configuration" to modify the preserve configuration items.
